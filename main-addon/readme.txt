StainedOfMind's GunBonsai eXtensions [Version 0.0.1]

Changelog
---------
Version 0.0.1:
-Initial release

Weapon Upgrades
---------------
Flak Shot: Configurable version of Submunitions. Works like the classic SoM 
version.
Shatterdrive: Configurable version of Submunitions, upgraded with a skill
tree.
Fragment Cascade: Shatterdrive Upgrade: Increase number of flaks.
Fusion Matrix: Shatterdrive Upgrade: Increase flak damage.
Decay Lattice: Shatterdrive Upgrade: Increase maximum flak time to live.

Player Upgrades
---------------
Ammuneux: Classic Bandoliers version from 0.10.6.
Pulse Reliquary: Max health booster.
Continuance Field: Powerup Duration Booster.
LOL Rawkitz: Dumb. Do not use.
Ordinance Array: Syphons excess ammo into a reserve to allow for greater ammo
capacity, whilst improving compatibility over the old Bandoliers method.
